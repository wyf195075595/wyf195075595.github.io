### vue 多页面 history 路由 配置

多页面打包配置

```js
pages: {
    index: {
      entry: "src/main.js",
      template: "public/index.html",
      filename: "index.html",
      title: "PMS index",
    },
    testPMS: {
      entry: "src/testPMS.js",
      template: "public/testPMS.html",
      filename: "testPMS.html",
      title: "PMS test",
    },
},
```

testPMS页面路由

```js
const router = new VueRouter({
  base: "/testPMS",
  mode: "history",
  routes: [
    {
      path: "/",
      redirect: "/repairSchedule",
    },
    {
      path: "/repairSchedule",
      name: "xxxx",
      component: IframeView,
    },
 ]
})
```

index 页面路由

```js
const router = new VueRouter({
  routes: [
    {
      path: "/",
      redirect: "/repairSchedule",
      meta: {
        keepAlive: false,
        sort: false,
      },
    },
      {
      path: "/login",

        keepAlive: false,

    {
      path: "/404",
      name: "error-404",
      meta: {
        keepAlive: false,
        sort: "z",
      },
      component: () =>
        import(/* webpackChunkName: "error-404" */ "@/views/error/404View.vue"),
    },
    {
      path: "*",
      redirect: "/404",
      meta: {
        keepAlive: false,
        sort: false,
      },
    },
  ],
  base: "/",
  mode: "history",
});
```

nginx   配置资源路径

```js
server {
 listen 1973;
 server_name 192.168.1.12;
 location /jysp {
   	root '/home/jysp/workspace/pms3';
   	index index.html;
	try_files $uri $uri/ /jysp/index.html;
 }
 location /testPMS {
    root '/home/jysp/workspace/pms3';
    index testPMS.html;
    try_files $uri $uri/ /jysp/testPMS.html;
 }
}
```

> **注意：服务器的根路径 /home/jysp/workspace/pms3，我们配置了 /jysp/。所以我们需要在 在 pms3 下新增 /pms3/jysp 文件夹，然后将 打包文件放到jysp下**

### 部署项目时避免上次缓存

在nginx 指定资源类型不缓存

```json
server {
    listen       18004;
    server_name  192.168.1.12;
    access_log logs/pmsWeb.access.log main;
    location / {
        #对html设置不缓存
        if ($request_filename ~* .*\.(?:html|htm)$)
        {
        	add_header Cache-Control "no-cache,no-store,must-revalidate";
        }
        root /home/jysp/workspace/pmsweb/;
            index index.html index.htm;
        }
}
```
